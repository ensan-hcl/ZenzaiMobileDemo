# ZenzaiMobile

ニューラルかな漢字変換システムである「Zenzai」をiOS / iPadOSで動作させるデモアプリケーションです。

Zenzaiについてはこちらをお読みください。

* [ニューラルかな漢字変換エンジン「Zenzai」をazooKey on macOSに搭載します](https://zenn.dev/azookey/articles/ea15bacf81521e)
